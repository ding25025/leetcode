# Array
## Definition 
- Contiguous area of memory
- Constant-time acess
## Space
| Type      | Add  | Remove |
| --------- | ---- | ------ |
| Beginning | O(n) | O(n)   |
| End       | O(1) | O(1)   |
| Meddle    | O(n) | O(n)   |

## Corner Case
- Empty sequence
- Sequence with 1 or 2 elements
- Sequence with repeated elements
- Duplicated values in the sequence
